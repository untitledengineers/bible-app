# Bible

A Bible application.

## Prerequisites

- [Node.js >=12.16.0](https://nodejs.org/en)
- [Yarn >=1.22.0](https://classic.yarnpkg.com)

## Installation

Install dependencies:

```
yarn global add expo-cli
yarn install
```

## Running

Test application:

```
yarn start
```
